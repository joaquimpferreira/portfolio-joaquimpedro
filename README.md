# Portfólio - Joaquim Pedro

Este é o meu **portfólio pessoal**, desenvolvido com **React** e **Tailwind CSS**. O projeto apresenta meus trabalhos e habilidades como desenvolvedor, com uma interface moderna e responsiva. Utiliza Vite para um ambiente rápido e eficiente, além de uma estrutura organizada para facilitar a manutenção e escalabilidade.

**Portfólio:** *https://joaquimpferreira.github.io/portfolio-joaquimpedro/*

## Processo de desenvolvimento: 

- **Processo Criativo** – Desenvolvi o layout do portfólio no Figma, planejando a estrutura visual web, mobile, cores e tipografia.

- **Configuração do Ambiente** – Iniciei o projeto com React e Vite, garantindo um desenvolvimento ágil.

- **Estilização com Tailwind CSS** – Implementei o design utilizando classes utilitárias para um estilo moderno e responsivo.

- **Criação dos Componentes** – Dividi o projeto em componentes reutilizáveis para melhor organização e manutenção.

- **Ajustes e Responsividade** – Testei e refinei a interface para garantir uma boa experiência em diferentes dispositivos.

- **Otimização e Deploy** – Realizei melhorias de performance e publiquei o projeto online.

## Dependências

- **EmailJs** - Contato via e-mail (sem uso de banco de dados)
- **SplineTool** - Manipulação elemento 3D 
- **React Icons** - Icones nativos
- **React Scroll** - Manipulação Scroll
